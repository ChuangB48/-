程式語言簡介
使用電腦就一定會用到程式，而程式是以程式語言撰寫的，各種系統軟體及應用軟體，都是運用程式語言開發的，因此學習程式設計是學習電腦重要的課程之一。電腦的硬體只認識機器語言(machine code)，是一串0與1的組合。雖然機器語言的執行效率最快，但是不容易撰寫也不容易閱讀。於是就有了高階語言(high-level language)的發展，如C/C++、Python、Java等。其較接近人類的語言，而高階語言需要透過編譯器(compiler)轉譯成機器語言後才能執行。
C/C++的歷史
C語言於1969年至1973年間，為了移植與開發UNIX作業系統，由丹尼斯·里奇與肯·湯普遜，以B語言為基礎，在貝爾實驗室設計、開發出來。二十世紀八十年代，C語言應用日漸廣泛。為了避免各開發廠商用的C語言的語法產生差異，美國國家標準局為C語言訂定了一套完整的國際標準語法，稱為ANSI C，作為C語言的標準。與此同時，國際標準化組織也接受該標準為國際標準。因此，ANSI C也同時被稱為ISO C。二十世紀八十年代至今的有關程式開發工具，一般都支援符合ANSI C的語法。
比雅尼·斯特勞斯特魯普博士在貝爾實驗室工作期間在20世紀80年代發明並實現了C++。起初，這種語言被稱作「C with Classes」（「包含『類』的C語言」），作為C語言的增強版出現。
第一支C++程式
撰寫程式前，注意以下幾點:
♦字母大小寫是不同的。
♦每個敘述以分號 ";" 結束。
♦使用一個空白鍵或多個空白都是一樣的，程式碼應適度加入空白行或自由換行。
♦副檔名為.cpp。pp指的是plus plus。
#include <iostream>
using namespace std;
int main(){
    cout << "hello, world" << endl;
    return 0;
}
